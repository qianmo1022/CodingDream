# 1. 数组上的方法，在js中的意义？
var a = []
a.push()
a.pop()

1. 创建对象字面量在v8的执行过程中相当于 new了一个构造函数，通常我们不去直接new构造函数，因为不够简洁直观
2. 因为a相当于new Array()得到的，那么该对象的隐式原型(_proto_) === Array.prototype(显示原型),v8在查找属性的时候会先在对象上查找，
如果找不到则会顺着对象的原型往上查找

new会创建一个对象，让对象的隐式原型等于显示原型，让this指向这个对象，返回这个对象
